# ImplementacionJavaFX

Proyecto desarrollado en Java utilizando JavaFX y Maven para la implementación de interfaces gráficas y la práctica de aplicaciones visuales en escritorio.

## Descripción

Esta aplicación está orientada a la implementación de componentes gráficos mediante JavaFX, utilizando Maven para la gestión del proyecto y dependencias.

El proyecto permite trabajar con ventanas, controles y estructuras básicas de aplicaciones gráficas en Java.

## Tecnologías utilizadas

- Java
- JavaFX
- Maven

## Requisitos

- JDK 17 o superior
- Maven instalado
- IDE compatible con JavaFX
